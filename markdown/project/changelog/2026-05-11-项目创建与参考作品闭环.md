# 2026-05-11 项目创建与参考作品闭环

## 已完成

### 创建项目流程
- 将“新建项目”重构为 3 步向导：
  1. 基础方向
  2. 内容设定
  3. 确认创建
- 题材输入从“下拉 + 芯片 + 自由输入同时并存”调整为更清晰的流程。
- 文风预设补充了明确规则说明，不再只是黑盒标签。

### AI 设定助手
- 新增 AI 世界观扩写。
- 新增 AI 写作偏好扩写。
- 两类扩写统一走当前配置的真实模型链路。
- 扩写结果支持替换与追加到最终字段。

### 参考作品能力
- 新增“参考作品识别”接口。
- 创建项目时可识别参考作品，并展示确认卡：
  - 作品名
  - 创作者
  - 载体
  - 简介
  - 风格特征
  - 世界特征
  - 叙事约束
  - 识别说明
- 创建后将确认卡结构化持久化到项目。
- 项目设置页可再次查看确认卡。
- 项目设置页可修改确认卡字段。
- 项目设置页可重新识别参考作品。
- 风格特征 / 世界特征 / 叙事约束已改成可增删的标签编辑。

### 生成链路
- 项目的 `reference_work` 已进入正文生成 prompt。
- 写作时会读取参考作品可迁移指导语，而不是只停留在创建表单。

### 其他修复
- “新建文件夹”提交流程补了更明确的状态与反馈链。
